# 티카투카 시뮬레이터

티카투카 미니게임의 현재 필드를 입력하고, Action별 예상 승률과 추천 수를 계산하는 브라우저 앱입니다.

## 저장 방식

보드 상태, 현재 주사위, 리롤/홀드 설정, 결과가 확정된 관찰 데이터는 현재 브라우저의 `localStorage`에 자동 저장됩니다. 매 판마다 내보내기할 필요는 없습니다.

상대 행동은 현재 판의 임시 관찰로만 쌓이고, `승리`, `패배`, `무승부` 중 하나를 눌렀을 때 확정 기록으로 저장됩니다. 테스트 조작은 결과 저장을 누르지 않으면 학습/관찰 데이터에 포함되지 않습니다.

`백업 내보내기`는 다른 기기로 옮기거나 데이터를 보관할 때 사용합니다.

## 로컬 실행

